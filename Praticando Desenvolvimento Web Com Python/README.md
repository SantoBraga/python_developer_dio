# Consumindo a API do Twitter com Python

## Consumindo a API do Twitter com Python: 

- dreamFilter, pacote simples divido em dois módulos: 

  - filter:
    - Combinar uma ou duas imagens, aplicando um filtro.
  
  - tools:
    - Abrir imagem.
    - Comparar se duas imagens são iguais.
